Github

The final part of the assignment was to upload my website to github. I had already done something similar in the course Software Developement so i new how to connect my folder to github. To start with i had to create a repository on github and copy the clone URL of the repository. Step 2 was to open the terminal on my computer and change the working directory to my local folder where my website it saved. Step 3 was to type the command "git clone https://github.com/OlafurMelville/olafurmelville.github.io.git" and press enter. Step 4 was to add the local folder to the repository by typing the command "git ."" Step 5 was to commit the folder by typing the command "git commit -m 'First commit'" into the terminal. And finally pushing the folder by typing the command "git push origin master".

Workload-0.5 Hours
