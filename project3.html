Assignment 3 - 3D scanning and printing

Our third assignment in the course Manufacturing processes was split into two assignments. The first was to use a design software to make something that could not be made subtractiveley and then 3D print your design. The second part of the project was to 3D scan something with 3D photogrammetry

Part 1 - Solidworks

I decided to use the design software Solidworks to design a coupling for a prosthetic knee. The coupling can not be made with a laser cutting device because of its round and deep edges. It is a very small object used to keep the prosthetic knee intact. Unfortunately i was using solidworks on the computers at school and didn't take any pictures while i was designing the coupling in solidworks. Now while i am writing this, i am not able to go to school and take pictures of the coupling because the school has been closed because of the covid-19 pandemic. Instead i will try and describe what i did in words with the help of some autocad drawings. I had access to autocad drawings of the part that helped me design the coupling.

I began by sketching a rectangle and revolving it 360 degrees. Then with the help of the chamfer tool in solidworks, cut out the edges to create the coupling. Finally i used the extruded cut tool to create the hole in the middle of the coupling. The final design looked like the picture below.

Part 2 - Fablab

After creating the coupling in Solidworks it was time to visit Fablab-Reykjav√≠k to 3D print the coupling. Before i could 3D print the coupling i first had to export the file into a slt file and import the file into Ultimaker Cura. Ultimaker Cura is an open source 3D printer slicing application, Ultimaker Cura works by slicing the user's model file into layers and generating a printer-specific g-code. Once finished, the g-code can be sent to the file printer for the manufacture of the physical object.

I imported the file to Ultimaker Cura, i did not need to make any major changes to the settings in Ultimaker Cura. I changed some settings so the printing would be faster. In the end the coupling would weigh one gramm and the printing would take an estimate of 18 minutes.
The 3D printing was a success, the picture below is of the coupling that was 3D printed.

Regard 3D

After some discussion in class, somebody said that the software Regard3D was an easy to use software for this sort of assignment, so i downloaded Regard3D. I then googled the key word "Regard3D tutorial" and found a video that helped me with this assignment Regard 3D tutorial, it was now time to start using the software.

After opening Regard 3D a window opens up.

In the left hand corner there is a button to create a new project, I created a new project called "3D scan" and then imported the pictures into regard by clicking the add picture set button in the left bottom corner.
Then i pressed the compute matches button and adjust the settings, the tutorial i watched said to move the sliders to the right but leave other settings as they were.
The next step was to triangulate the matches. Start off by selecting the matches and then press the triangulation button, another window pops up that shows the triangulation settings. You dont need to change any settings just press ok.
The pount cloud that the triangulation produced was ok. To make the model even clearer and denser. Select the triangulation and create a dense pointcloud, i used the same settings the tutorial used.
The dense pointcloud produced a better image of the mug. The last step was to create a surface and connect the points in the point cloud together. The tutorial said to stick with the same settings that pop up when you click on "Create Surface" so that is what i did. After doing all the steps the tutorial told me to do, the final 3D model came out ok.
